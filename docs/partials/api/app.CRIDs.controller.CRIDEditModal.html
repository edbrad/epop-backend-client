CRIDEditModal
controller in module app.CRIDs

Description

Controller for modal View to add USPS CRID to a given Mail Owner

Dependencies

  • lbServices

    (Loopback API service)

Usage

CRIDEditModal($q:, $scope:, $modalInstance:, logger:, data:, CRID:, MailOwner:);

Parameters

ParamTypeDetails
$q:object

Angular asychronous process object

$scope:object

Angular Scope object

$modalInstance:object

Angular Bootsrap modal instance object

logger:object

Internal Logging Service

data:array

Genertic data (button text)

CRID:object

Selected CRID

MailOwner:object

Selected Mail Owner

Methods

  • cancel()

    perform 'CANCEL' processing (dismiss the Modal)

  • getCRID()

    collect selected CRID and asociated Mail Owner from database to display in edit form

  • getMailOwners()

    collect all available Mail Owners, to select which is to be associated with the new CRID

  • update()

    perform 'UPDATE' processing (update the CRID in the database, then close the dialog)
