deleteImportStatement

Description

Delete a stored statement and deduct piece & postage counts from total

Usage

deleteImportStatement(id, pieces, postage);

Parameters

ParamTypeDetails
idstring
  • The eDoc Statement Id
piecesnumber
  • The number of pieces represented in the statement (to be deducted from total)
postagenumber
  • The amount of postage represented in the statement (to be deducted from total)
